# data-pipline

## Installation

```python
pip install .
```

## Setup

From inside the project directory
```
$ data_pipeline --setup
```

From everywhere else
```
$ data_pipeline --setup --project <project_dir>
```

## Configure


From inside the project directory
```
$ data_pipeline --configure
```

From everywhere else
```
$ data_pipeline --configure --project <project_dir>
```
